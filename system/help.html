Frequently Asked Questions (FAQ)

Have questions? We've got answers! Here are some frequently asked questions about Web XP:

  1. How can I change the wallpaper in Web XP?

    2. Open Start Menu, navigate to personalization, click on change wallpaper. Choose from default stock wallpapers or add your own from a direct URL

  2. Can I install or make applications for Web XP?

    3. Currently Web XP can't support Any App Store. But you can develop your own programs for Web XP and mail us, our team will review your web app and if they find it useful, it will be added in Web XP in future releases.

  3. How do I create a new user account in Web XP?

    4. Open Start Menu, navigate to User Accounts & Control - Choose options from changing your name, profile picture or add a new account

  4. Is there a way to backup and restore my Web XP settings?

    5. Currently the data on Web XP is stored in your Web Browser as persistent cache storage, and all your data will be erased if you clear the site data. Unfortunately there are no options to backup your data, but however, you can always download your files to your local OS. Check out our Privacy Policy.
